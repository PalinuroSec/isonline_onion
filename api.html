This service provide the easiest API possible.
Also this page is simple as possible.

Get server status:

Request:
  GET ?a=[hidden service link]&format=json

  GET => request type
    a: the hidden service link (can be with or without "HTTP://" but it must be a valid .onion domain)
    format: the response format (only json avviable right now)
Response: 
  JSON Object
  { 
    "url": the request's link/a argument
    "status": an HTTP Protocol status (possible status: 200,302,403,404,502,503)
  }

Read this for a description of HTTP status code


So if the request is something like this:
http://localhost:8888/?a=ol56t3xahrpk2b62.onion&format=json

The response will be something like:
{"url":"ol56t3xahrpk2b62.onion","status":"200"}
